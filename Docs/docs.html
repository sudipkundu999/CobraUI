Alerts

Alerts are used to communicate the user about any event

This is a Failure alert!

This is a Warning alert!

This is a General alert!

            <i class="fas fa-check fa-2x"></i>        
        </div>
        <div class="alert-text">
            This is a Success alert!
        </div>
    </div>

Badge

Badges are small components placed at the top or bottom right of the parent component to highlight an items status using colors or numbers or both


Badge on Icons

12
New 

    <div class="badge">
        <i class="fas fa-bell fa-3x"></i>
        <span class="badge-color"></span>
    </div>

    <div class="badge">
        <i class="fas fa-heart fa-3x"></i>

    </div>
    
    <div class="badge">
        <i class="fas fa-envelope fa-3x"></i>
